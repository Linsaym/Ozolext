# Ozolext (Ozon Lev Extension)

> Расширение для браузера + AutoHotkey v2 скрипты для стабильной работы с Turbo PVZ Ozon.

## 📦 Что внутри

- **Браузерное расширение** (минималистичная архитектура) — добавляет фиксированные управляющие кнопки в интерфейс Ozon.
- **Скрипты AutoHotkey v2** — обеспечивают автоматизацию и связку с горячими клавишами.
- **Установщик** (в архиве релиза) — для быстрой настройки AHK-части.

## 🚀 Быстрый старт

1. Скачайте последний [Release](https://github.com/Linsaym/Ozolext/releases)
2. Распакуйте архив.
3. Установите AHK и запустите скрипты через установщик
4. Загрузите [расширение в браузере (режим разработчика)](chrome://extensions/)
5. Готово — кнопки появятся на страницах Ozon Turbo PVZ
6. Настройте скрипты под себя и свой монитор

## 🛠 Технологии

- AutoHotkey v2
- JavaScript (Manifest V3)
- HTML/CSS
